# anyrun 启动器配置

| 文件 | 前缀 | 说明 | 使用示例 |
| ------ | ------ | ------ | --------- |
| `applications` | 无 | 应用程序搜索，启用电源操作 | 直接输入应用名 |
| `actions` | 无 | 自定义动作（需在配置中定义） | 直接输入定义的动作名 |
| `rink` | 无 | 计算器 | 直接输入表达式 |
| `websearch` | `?` | 网页搜索 | `? rust async` |
| `translate` | `/t` | 翻译 | `/t zh-CN>en 你好` |
| `symbols` | `/s` | 符号复制 | `/s arrow` |
| `shell` | `>` | Shell 命令执行 | `> htop` |
| `ssh-pattern` | `//` | SSH 连接 | `//>kitty test` |
| `menu-bar` | `/m` | 菜单栏 | `/m a/b` |
